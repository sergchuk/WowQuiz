# Сайт для курсовой работы по дисциплине Проектирование и дизайн ИС

## Курсовая работа Word
* Титульный лист
* Содержание с нумерацией страниц
* Введение с целью и задачами
* Раздел 1: Теория
* Раздел 2: Практика
* Заключение
* Список литературы
* Приложение А с описанием кода

## Сайт WowQuiz

Браузерная викторина на чистом HTML, CSS и JavaScript без фреймворков.  
Результаты сохраняются в `localStorage` браузера — регистрация не требуется.

### Страницы

| Файл | Назначение |
|---|---|
| `index.html` | Главная — приветствие, выбор категории, последние результаты |
| `quiz.html` | Викторина — 3 категории по 10 вопросов с 4 вариантами ответа |
| `leaderboard.html` | Таблица лидеров — все результаты с фильтрами и статистикой |
| `profile.html` | Профиль — прогресс по категориям, 8 достижений, история игр |
| `about.html` | О проекте — правила игры, описание категорий, FAQ |

### Категории вопросов

* **Общие знания** — история, география, литература, базовая наука
* **Наука** — биология, физика, химия, астрономия
* **Технологии** — программирование, IT, интернет, известные компании

### Функциональность

* Выбор категории на главной странице
* Прогресс-бар и счётчик очков в реальном времени
* Подсветка правильного и неправильного ответа
* Экран результата с круговой диаграммой процента и текстовой оценкой
* Сохранение всех результатов в `localStorage`
* Таблица лидеров с сортировкой по проценту и фильтрами по категориям
* Профиль с прогрессом по каждой категории и системой достижений (8 штук)
* Страница «О проекте» с инструкцией и аккордеон-FAQ
* Единая навигация на всех страницах

### Технологии

* HTML5, CSS3 (Grid, Flexbox, CSS-переменные)
* Vanilla JavaScript (без библиотек)
* Web Storage API (`localStorage`)
* URL Params API (`URLSearchParams`)
